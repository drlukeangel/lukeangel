# Blog images

Drop blog images here, then reference them in posts with `/blog/your-file.jpg`.

Recommended sizes:
- Cover image: 1200×630 px (Open Graph standard) — JPG, ~200KB
- Inline images: max 1600 px wide — JPG or PNG, ~200KB

These are served as-is. If you want auto-optimization, put images in
`src/assets/blog/` and use the `<Image>` component in an .mdx post.
See `src/content/blog/example-with-images.mdx` for a sample.
